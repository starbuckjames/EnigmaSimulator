# EnigmaSimulator
A Simulator and Solver for the Enigma Cipher Machine from WWII.  Was was for a school project, not intended for any commerical purpose.  If you want to use it for anything, contact me first.

EnigmaSimulator3.py simulates the 3-rotor German Army/Air Force Enigma machine with plugboard.  It has a fully implemented "double ratchet" simulation, and the plugboard, rotors and the starting ratchet positions on the rotors are changable.

EnigmaSolverGPU7.py uses CUDA and Numba so python can work with GPUs.
